Завдання :
За основу взяти минуле завдання по роботі з базою даних продажів. Реалізувати
наступні речі:
1. Уникнути SQL ін'єкцій при виконані запитів до БД.
2. Перенести класи для доступу до БД в окрему бібліотеку (Class Library).
3. Створити WPF проект, який надасть можливість переглядати список клієнтів,
продавців та продажів.
4. Перенести рядок підключення до БД в конфігураційний файл (App.config).
